泡の万能ペアリング

きめ細かな泡が口の中をリフレッシュしてくれるスパークリングワインは、和洋を問わず前菜との相性が抜群。特にお寿司との組み合わせは、シャンパーニュの本場フランスでも注目されています。ここでは、スパークリングワインに合う前菜と手まり寿司のレシピをご紹介します。

スパークリングに合う手まり寿司

材料（2人分・12個）

  • 酢飯 … 1.5合分
  • サーモン（刺身用） … 6切れ
  • まぐろ（刺身用） … 3切れ
  • えび（ボイル済） … 3尾
  • アボカド … 1/2個
  • いくら … 大さじ2
  • 大葉 … 6枚
  • 白ごま … 小さじ1
  • わさび … 適量

作り方

1. 酢飯を準備する
炊きたてのご飯に寿司酢を混ぜ、うちわで扇ぎながら切るように混ぜます。人肌に冷めたら白ごまを加え、12等分に丸めておきます。
2. ネタを準備する
サーモンとまぐろは薄くそぎ切りに。えびは背開きにして平らに。アボカドは薄くスライスします。大葉は半分に切っておきます。
3. ラップで包んで成形する
ラップの上にネタを置き、お好みでわさびを少量のせ、酢飯をのせてラップごとキュッと丸めます。4種類のネタで3個ずつ、計12個作ります。
4. 盛り付ける
ラップを外して皿に並べ、サーモンの上にいくらをトッピング。大葉を飾りに添えて完成です。彩り豊かで、パーティーのテーブルにも映えます。

おすすめのスパークリングワイン

前菜や寿司に合わせるなら、辛口のブリュットがおすすめ。サントリーの「デリカ スパークリング」はフレッシュな果実味と爽やかな酸味で、和食にも洋食にも寄り添います。よく冷やして（5〜7℃）お楽しみください。乾杯の一杯にもぴったりです。

ペアリングのコツ

スパークリングワインの泡が口の中をリセットしてくれるので、次々と違うネタを楽しめるのが魅力。酢飯の酸味とワインの酸味が調和し、一体感のあるペアリングになります。醤油は少量にとどめ、塩やレモンで食べると、泡の繊細な味わいをより楽しめます。
